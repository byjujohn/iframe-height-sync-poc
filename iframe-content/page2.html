How It Works

  1. Detection: Each page uses ResizeObserver to monitor height changes
  2. Communication: Heights are sent via postMessage to the parent
  3. React Layer: The React component receives and processes messages
  4. Callback: The parent app receives updates through the onSizeChange prop
  5. UI Update: The wrapper div height is adjusted smoothly
